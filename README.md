Data Analytics Project
Overview

This project demonstrates an end-to-end data analytics workflow, from data loading and exploratory data analysis (EDA) to database querying and business intelligence reporting. The goal is to extract actionable insights and present them through interactive dashboards and clear documentation.

Dataset

The dataset contains structured data relevant to business/customer analysis

Includes multiple attributes used for analysis, filtering, and aggregation

Data quality issues such as missing values and inconsistencies were addressed during cleaning

Tools & Technologies

Python (Pandas, NumPy, Matplotlib/Seaborn) for EDA and data cleaning

SQL (PostgreSQL / MySQL / SQL Server) for querying and analysis

Power BI for dashboard development

Gamma for presentation (PPT) creation

Project Steps

Load and inspect the dataset

Perform exploratory data analysis (EDA)

Clean and preprocess the data

Load cleaned data into SQL databases

Write and execute SQL queries for insights

Build interactive Power BI dashboards

Create a detailed analytical report

Summarize insights in a presentation

Dashboard

Interactive Power BI dashboard with key KPIs

Visualizes trends, patterns, and performance metrics

Designed for business users and decision-makers

Results

Identified key trends and insights from the data

Improved understanding of patterns through SQL and visual analytics

Delivered insights in dashboard, report, and presentation formats

How to Run

Clone the repository

Install required Python libraries

Run the EDA and data cleaning scripts

Import the cleaned data into your SQL database

Execute SQL queries provided

Open the Power BI file to explore the dashboard# customer_behavior_analysis
